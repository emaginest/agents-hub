Introduction

Get started with Agents Hub, an advanced agent orchestration framework.

New: Version 0.1.2 released! New WebSearchTool and PlaywrightScraperTool added. Check out what's new.

Agents Hub is a revolutionary framework for creating, managing, and orchestrating intelligent agent workforces. It provides a comprehensive solution for building multi-agent systems with emergent intelligence capabilities.

Why Agents Hub?

Agents Hub helps you build sophisticated AI agent systems with multiple specialized agents working together. The framework provides:

  • A unified interface for working with multiple LLM providers (OpenAI, Claude, Gemini, Ollama)
  • Advanced cognitive architecture with metacognitive capabilities
  • Flexible memory systems with multiple backend options
  • Dynamic orchestration for agent collaboration
  • Comprehensive tool system for extending agent capabilities
  • Built-in moderation and safety mechanisms
  • Monitoring and analytics integration

General Architecture

At its core, Agents Hub follows a modular architecture:

  • Agent System: Flexible agent creation and management with specialized capabilities
  • Cognitive Architecture: Multi-layered cognitive system inspired by human cognition
  • LLM Integration: Support for multiple LLM providers with a unified interface
  • Memory System: Sophisticated memory management with multiple backend options
  • Orchestration: Dynamic task routing and agent collaboration
  • Knowledge Management: Advanced RAG capabilities with PostgreSQL/pgvector
  • Tool System: Extensible tool framework for agent capabilities
  • Coding System: Specialized agents and tools for software development
  • Moderation System: Content filtering and safety mechanisms
  • Monitoring System: Performance tracking and analytics

Quick Example

Here's a simple example of creating an agent with Agents Hub:

from agents_hub import Agent
from agents_hub.llm.providers import OpenAIProvider

# Initialize LLM provider
llm = OpenAIProvider(api_key="your-openai-api-key")

# Create an agent
agent = Agent(
    name="assistant",
    llm=llm,
    system_prompt="You are a helpful assistant."
)

# Use the agent
response = await agent.run("What is the capital of France?")
print(response)
