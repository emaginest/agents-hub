Cognitive Architecture

Understand the multi-layered cognitive system inspired by human cognition.

Overview

The cognitive architecture in Agents Hub is inspired by human cognition, providing agents with advanced reasoning, metacognition, and learning capabilities. This architecture enables agents to process information more effectively, reflect on their own thinking, and improve over time.

The cognitive architecture consists of five main components:

  • Perception: Processes input and extracts features
  • Working Memory: Stores active information
  • Reasoning: Applies appropriate reasoning mechanisms
  • Metacognition: Reflects on the reasoning process
  • Learning: Updates knowledge and strategies based on experience

Components

Perception

The perception layer processes input text and extracts features:

  • Feature Extraction: Identifies entities, keywords, and patterns
  • Context Recognition: Recognizes context from input and history
  • Attention Mechanism: Focuses on relevant parts of the input
from agents_hub.cognitive import Perception

# Create perception layer
perception = Perception(
    feature_extraction=True,
    context_recognition=True,
    attention_mechanism=True,
    max_features=10,
)

# Process input
result = await perception.process(
    input_text="If all birds can fly, and penguins are birds, can penguins fly?",
    context={},
)

Working Memory

The working memory system stores and manages active information:

  • Capacity Management: Limits the amount of information (Miller's Law: 7 ± 2 items)
  • Decay: Information fades over time if not reinforced
  • Chunking: Groups related information to increase effective capacity
  • Prioritization: Focuses on the most important information
from agents_hub.cognitive import WorkingMemory

# Create working memory
memory = WorkingMemory(
    capacity=7,  # Miller's Law: 7 ± 2 items
    decay_rate=0.1,
    chunking_enabled=True,
    prioritization_enabled=True,
)

# Update memory with new information
memory_state = await memory.update(
    perception_result=perception_result,
    context={},
)

Reasoning

The reasoning layer applies different reasoning mechanisms:

  • Deductive Reasoning: Drawing conclusions from general principles
  • Inductive Reasoning: Forming generalizations from specific observations
  • Abductive Reasoning: Finding the most likely explanation for observations
  • Analogical Reasoning: Applying knowledge from one domain to another
  • Causal Reasoning: Understanding cause-and-effect relationships
from agents_hub.cognitive import ReasoningManager

# Create reasoning manager
reasoning = ReasoningManager(
    enabled_mechanisms=["deductive", "inductive", "abductive"],
    default_mechanism="deductive",
)

# Apply reasoning
reasoning_result = await reasoning.reason(
    memory_state=memory_state,
    context={"reasoning_mechanism": "deductive"},
)

Metacognition

The metacognitive layer reflects on the reasoning process:

  • Self-Monitoring: Evaluates the quality of reasoning
  • Confidence Assessment: Estimates confidence in conclusions
  • Error Detection: Identifies potential errors or biases
  • Strategy Selection: Chooses appropriate reasoning strategies
  • Reflection: Considers alternative approaches
from agents_hub.cognitive import Metacognition

# Create metacognition layer
metacognition = Metacognition(
    reflection_depth=2,
    confidence_threshold=0.7,
)

# Reflect on reasoning
metacognition_result = await metacognition.reflect(
    reasoning_result=reasoning_result,
    context={},
)

Learning

The learning layer updates knowledge and strategies based on experience:

  • Knowledge Updating: Incorporates new information
  • Strategy Refinement: Improves reasoning strategies
  • Pattern Recognition: Identifies recurring patterns
  • Feedback Integration: Learns from feedback
from agents_hub.cognitive import Learning

# Create learning layer
learning = Learning(
    learning_rate=0.1,
    feedback_integration="immediate",
)

# Update knowledge and strategies
learning_result = await learning.update(
    metacognition_result=metacognition_result,
    context={},
)

Cognitive Agent

The CognitiveAgent class integrates the cognitive architecture with the base Agent:

from agents_hub import CognitiveAgent
from agents_hub.llm.providers import ClaudeProvider
from agents_hub.cognitive import CognitiveArchitecture

# Initialize LLM provider
llm = ClaudeProvider(api_key="your-anthropic-api-key")

# Create cognitive architecture
cognitive_architecture = CognitiveArchitecture(
    metacognition_config={
        "reflection_depth": 2,
        "confidence_threshold": 0.7,
    },
    reasoning_config={
        "enabled_mechanisms": ["deductive", "inductive", "abductive"],
        "default_mechanism": "deductive",
    },
)

# Create cognitive agent
agent = CognitiveAgent(
    name="cognitive_agent",
    llm=llm,
    cognitive_architecture=cognitive_architecture,
    system_prompt="You are a thoughtful assistant with advanced reasoning capabilities.",
    cognitive_config={
        "reasoning_trace_enabled": True,
        "metacognition_enabled": True,
        "learning_enabled": True,
    },
)

# Use the agent
response = await agent.run(
    "What are the main causes of climate change?",
    context={"reasoning_mechanism": "deductive"}
)

Reasoning Trace

One of the key features of the cognitive architecture is the reasoning trace, which provides transparency into the agent's thinking process:

# Access the reasoning trace after running the agent
print(agent.last_reasoning_trace)

# Example output:
"""

- Identified topic: climate change
- Identified task: explain causes
- Context: scientific explanation

Reasoning (Deductive):
1. Climate change refers to long-term shifts in temperatures and weather patterns.
2. These shifts can be natural, but since the 1800s, human activities have been the main driver.
3. The primary human activity affecting climate is burning fossil fuels.

Metacognition:
- Confidence: High (scientific consensus)
- Potential biases: None identified
- Alternative perspectives considered: Natural climate cycles
- Areas for improvement: Could provide more quantitative data

Final conclusion:
The main causes of climate change are human activities, primarily:
1. Burning fossil fuels (coal, oil, gas) which releases greenhouse gases
2. Deforestation which reduces CO2 absorption
3. Industrial processes and agriculture which release methane and other greenhouse gases
"""

Benefits of Cognitive Architecture

Using the cognitive architecture provides several benefits:

  • Improved Reasoning: More structured and effective reasoning processes
  • Transparency: Clear reasoning traces for explainability
  • Self-Improvement: Agents can learn and improve over time
  • Error Detection: Metacognitive capabilities help identify and correct errors
  • Adaptability: Different reasoning mechanisms for different types of problems

Use Cases

The cognitive architecture is particularly useful for:

  • Complex Problem Solving: Tasks requiring multi-step reasoning
  • Educational Applications: Explaining concepts and guiding learning
  • Research Assistance: Analyzing information and drawing conclusions
  • Decision Support: Evaluating options and making recommendations
  • Critical Thinking: Identifying biases and logical fallacies
