Frequently Asked Questions

Find answers to common questions about Agents Hub.

General Questions

What is Agents Hub?

Agents Hub is an advanced agent orchestration framework that allows you to create, manage, and orchestrate intelligent agent workforces. It provides a comprehensive solution for building multi-agent systems with emergent intelligence capabilities.

How does Agents Hub compare to LangChain?

While both frameworks provide tools for building LLM-powered applications, Agents Hub focuses specifically on multi-agent orchestration with advanced cognitive capabilities. Key differences include:

  • Agents Hub has a built-in cognitive architecture inspired by human cognition
  • Agents Hub provides more sophisticated agent orchestration mechanisms
  • Agents Hub includes specialized workforces for specific domains like software development
  • Agents Hub has integrated moderation and safety mechanisms

What license is Agents Hub released under?

Agents Hub is released under the MIT License, which allows for both personal and commercial use with minimal restrictions.

Technical Questions

What Python version is required?

Agents Hub requires Python 3.9 or higher.

Which LLM providers are supported?

Agents Hub supports multiple LLM providers:

  • OpenAI (GPT-4o, GPT-4, GPT-3.5-Turbo)
  • Anthropic (Claude 3 Opus, Sonnet, Haiku)
  • Google (Gemini models)
  • Ollama (local models)
  • Azure OpenAI
  • Hugging Face

What memory backends are supported?

Agents Hub supports multiple memory backends:

  • PostgreSQL (with pgvector)
  • Redis
  • In-memory
  • File-based

Does Agents Hub support async/await?

Yes, Agents Hub is built with asyncio and supports async/await throughout the framework.

Usage Questions

Do I need API keys for the LLM providers?

Yes, you need API keys for the cloud-based LLM providers (OpenAI, Anthropic, Google). However, you can use Ollama for local LLMs without an API key.

How much does it cost to use Agents Hub?

Agents Hub itself is free and open source. However, you will need to pay for the API usage of the LLM providers you choose to use. The cost depends on the provider, model, and usage volume.

Is Agents Hub ready for production use?

Agents Hub is currently in early development (version 0.1.0) and may not be fully stable for production use. We recommend using it for research, prototyping, and non-critical applications while we work towards a stable 1.0 release.

How do I deploy Agents Hub in production?

Agents Hub can be deployed like any Python application. We recommend using Docker for containerization and a production-ready ASGI server like Uvicorn or Hypercorn behind a reverse proxy like Nginx.

Support Questions

How do I report bugs?

You can report bugs by creating an issue on GitHub. Please include detailed steps to reproduce the bug, expected behavior, and actual behavior.

How do I request features?

You can request features by creating an issue on GitHub with the "feature request" label. Please describe the feature in detail and explain why it would be valuable.

Where can I get help?

You can get help through the following channels:

  • GitHub Discussions for general questions and discussions
  • GitHub Issues for bug reports and feature requests
