Building Cognitive Agents

Learn how to create advanced cognitive agents with metacognition capabilities, multiple reasoning mechanisms, and learning abilities.

Introduction to Cognitive Agents

Cognitive agents are an advanced type of agent in Agents Hub that go beyond simple prompt-response interactions. They incorporate a multi-layered cognitive architecture inspired by human cognition, enabling them to:

  • Reflect on their own thinking process (metacognition)
  • Employ multiple reasoning strategies
  • Learn from past interactions
  • Manage their own cognitive resources
  • Adapt to different tasks and contexts

In this tutorial, you'll learn how to create and configure cognitive agents for complex tasks that require sophisticated reasoning.

Prerequisites

Before starting this tutorial, you should:

  • Have Agents Hub installed (pip install agents-hub)
  • Be familiar with creating basic agents (see the Quickstart guide)
  • Have API keys for the LLM providers you want to use (Claude or GPT-4 recommended for cognitive agents)

Creating a Cognitive Agent

Let's start by creating a basic cognitive agent:

from agents_hub import CognitiveAgent
from agents_hub.llm.providers import ClaudeProvider
from agents_hub.cognitive import CognitiveArchitecture

# Initialize LLM provider
llm = ClaudeProvider(api_key="your-api-key")

# Create cognitive architecture
cognitive_architecture = CognitiveArchitecture(
    metacognition_enabled=True,
    reasoning_mechanisms=["deductive", "inductive", "abductive"],
    learning_enabled=True
)

# Create cognitive agent
agent = CognitiveAgent(
    name="cognitive_assistant",
    llm=llm,
    cognitive_architecture=cognitive_architecture,
    system_prompt="You are a cognitive assistant with advanced reasoning capabilities."
)

# Use the agent
response = await agent.run("What are the potential implications of quantum computing on cryptography?")

Configuring the Cognitive Architecture

The cognitive architecture is the core component that gives cognitive agents their advanced capabilities. You can configure various aspects of the cognitive architecture:

Metacognition

Metacognition allows the agent to reflect on its own thinking process, monitor its confidence, and adjust its approach when needed:

cognitive_architecture = CognitiveArchitecture(
    metacognition_config={
        "reflection_depth": 2,  # How many levels of reflection to perform
        "confidence_threshold": 0.7,  # Threshold for triggering deeper reflection
        "self_correction_enabled": True  # Whether the agent can correct itself
    }
)

Reasoning Mechanisms

Cognitive agents can use multiple reasoning strategies:

cognitive_architecture = CognitiveArchitecture(
    reasoning_config={
        "mechanisms": ["deductive", "inductive", "abductive"],
        "strategy_selection": "adaptive",  # Can be "fixed" or "adaptive"
        "chain_of_thought": True  # Enable step-by-step reasoning
    }
)

Learning Capabilities

Cognitive agents can learn from past interactions:

cognitive_architecture = CognitiveArchitecture(
    learning_config={
        "episodic_memory_enabled": True,  # Remember specific interactions
        "semantic_memory_enabled": True,  # Build knowledge structures
        "learning_rate": 0.5  # How quickly to incorporate new information
    }
)

Advanced Example: Research Assistant

Let's create a more advanced cognitive agent that acts as a research assistant:

from agents_hub import CognitiveAgent
from agents_hub.llm.providers import ClaudeProvider
from agents_hub.cognitive import CognitiveArchitecture
from agents_hub.tools.standard import WebSearchTool, CalculatorTool
from agents_hub.memory import PostgreSQLMemory

# Initialize tools
web_search = WebSearchTool()
calculator = CalculatorTool()

# Initialize memory
memory = PostgreSQLMemory(
    connection_string="postgresql://user:password@localhost:5432/agents_hub",
    table_name="research_agent_memory"
)

# Create cognitive architecture
cognitive_architecture = CognitiveArchitecture(
    metacognition_config={
        "reflection_depth": 2,
        "confidence_threshold": 0.7,
        "self_correction_enabled": True
    },
    reasoning_config={
        "mechanisms": ["deductive", "inductive", "abductive"],
        "strategy_selection": "adaptive",
        "chain_of_thought": True
    },
    learning_config={
        "episodic_memory_enabled": True,
        "semantic_memory_enabled": True,
        "learning_rate": 0.5
    }
)

# Create cognitive agent
research_agent = CognitiveAgent(
    name="research_assistant",
    llm=ClaudeProvider(api_key="your-api-key"),
    cognitive_architecture=cognitive_architecture,
    tools=[web_search, calculator],
    memory=memory,
    system_prompt="""You are an advanced research assistant with cognitive capabilities.
    You can search the web, perform calculations, and reason through complex problems.
    When analyzing research questions:
    1. Break down complex questions into smaller components
    2. Consider multiple perspectives and approaches
    3. Evaluate the reliability of information sources
    4. Synthesize information into coherent insights
    5. Acknowledge limitations and uncertainties in your analysis"""
)

# Use the agent
response = await research_agent.run(
    "Analyze the potential impact of machine learning on healthcare diagnostics over the next decade. 
    Consider technological, ethical, and economic factors."
)

Accessing Cognitive Traces

One of the powerful features of cognitive agents is the ability to access their reasoning traces:

# Run the agent
response = await agent.run("What are the ethical implications of AI in healthcare?")

# Access the reasoning trace
reasoning_trace = agent.last_reasoning_trace

# Print the metacognitive reflections
print("Metacognitive reflections:")
for reflection in reasoning_trace.metacognition.reflections:
    print(f"- {reflection}")

# Print the reasoning steps
print("\nReasoning steps:")
for step in reasoning_trace.reasoning.steps:
    print(f"- {step}")

# Print confidence assessment
print(f"\nConfidence: {reasoning_trace.metacognition.confidence}")

Best Practices for Cognitive Agents

Here are some best practices for working with cognitive agents:

LLM Selection

Cognitive agents require powerful LLMs to fully leverage their capabilities:

  • Recommended: Claude 3 Opus, GPT-4
  • Acceptable: Claude 3 Sonnet, GPT-3.5-Turbo
  • Not recommended: Smaller models with limited reasoning capabilities

System Prompts

Design system prompts that encourage metacognition and careful reasoning:

  • Explicitly instruct the agent to reflect on its thinking
  • Encourage consideration of multiple perspectives
  • Ask the agent to evaluate its confidence
  • Prompt the agent to identify potential biases or limitations

Resource Management

Cognitive agents can be more resource-intensive:

  • Be mindful of token usage, as metacognition increases prompt size
  • Consider using a tiered approach: start with simpler agents and escalate to cognitive agents for complex tasks
  • Use appropriate caching strategies to avoid redundant computations

Conclusion

Cognitive agents represent a significant advancement in agent capabilities, enabling more sophisticated reasoning, self-reflection, and learning. By properly configuring the cognitive architecture and providing appropriate system prompts, you can create agents that approach problems with human-like cognitive strategies.

For more examples of cognitive agents in action, check out the Examples page.
