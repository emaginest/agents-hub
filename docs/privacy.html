Privacy Policy

This privacy policy explains how we handle data in the Agents Hub framework.

Last Updated: April 14, 2025

Introduction

Agents Hub is an open-source framework for creating, managing, and orchestrating intelligent agent workforces. This privacy policy explains how the framework handles data and what you should know when using it.

Data Collection

As an open-source framework, Agents Hub itself does not collect any data from its users. However, when you use Agents Hub with third-party LLM providers (like OpenAI, Anthropic, Google), those providers may collect data according to their own privacy policies.

Third-Party Services

Agents Hub integrates with various third-party services:

  • LLM Providers: When you use Agents Hub with LLM providers like OpenAI, Anthropic, or Google, your prompts and data may be sent to these providers. Please refer to their respective privacy policies for information on how they handle your data.
  • Database Systems: If you use the memory system with PostgreSQL or other database backends, your data will be stored according to your database configuration.
  • Monitoring Services: If you enable monitoring integrations, usage data may be sent to those services.

Data Storage

Agents Hub provides memory backends that can store conversation history and other data. By default, this data is stored locally or in the database you configure. You are responsible for securing this data and ensuring it complies with relevant regulations.

Data Security

We recommend the following security practices when using Agents Hub:

  • Store API keys securely using environment variables or a secure key management system
  • Implement proper access controls for any databases used with Agents Hub
  • Regularly update Agents Hub and its dependencies to get the latest security fixes
  • Use the moderation system to filter sensitive content

User Responsibilities

As a user of Agents Hub, you are responsible for:

  • Complying with the terms of service of any third-party services you use with Agents Hub
  • Ensuring your use of Agents Hub complies with relevant laws and regulations
  • Securing any sensitive data processed by your implementation
  • Obtaining appropriate consent if you collect personal data from end users

Changes to This Policy

We may update this privacy policy from time to time. We will notify users of any significant changes by updating the date at the top of this policy.

Contact Us

If you have any questions about this privacy policy, please contact us at privacy@emaginest.com.
